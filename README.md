## рҹ“Ң About the Course

This course explores how **Artificial Intelligence (AI)** is transforming computer networks from both:

- the **operational / defensive perspective** (optimization, automation, monitoring)
- the **offensive / adversarial perspective** (attacks, evasion, exploitation)

It provides a unified view of modern networked systems where AI is both:

- a **tool for improvement**
- and a **surface for attack**

Topics include:

- Machine Learning for networking
- AI-driven cybersecurity
- LLM-assisted network operations
- Adversarial machine learning
- Trustworthy and safe AI deployment in networked environments

## рҹ§  Course Themes

The course is structured around three major pillars:

### 1. ML for Networking
- Traffic classification
- Network anomaly detection
- Telemetry and observability analytics
- Traffic forecasting
- Routing and congestion control
- Resource management in dynamic environments

### 2. AI for Cybersecurity
- Intrusion detection systems (IDS)
- DDoS detection and mitigation
- Malware analysis and detection
- Cyber threat intelligence
- Adversarial machine learning in security contexts

### 3. LLMs for Network Automation
- Log analysis and summarization
- AI-assisted troubleshooting
- Configuration understanding and generation
- Retrieval-augmented assistants for network operations
- Human-in-the-loop operational copilots

---

## рҹӣЎпёҸ Cross-Cutting Theme: Trustworthy AI

A key focus throughout the course is ensuring **safe and reliable AI deployment**, including:

- Robustness to adversarial inputs
- Model explainability and interpretability
- Calibration and uncertainty awareness
- Governance and policy constraints
- Human oversight and bounded autonomy
