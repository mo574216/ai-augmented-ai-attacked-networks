# AI-Augmented and AI-Attacked Computer Networks

This repository contains the course website and teaching materials for the graduate-level course:

**AI-Augmented and AI-Attacked Computer Networks**

The course explores how artificial intelligence is transforming computer networks from both the **operational/defensive** side and the **offensive/adversarial** side. It covers machine learning for networking, AI for cybersecurity, LLMs for network automation, adversarial machine learning, and trustworthy AI for networked systems.

This repository is designed to support a GitHub Pages course website.

---

## Course Themes

The course is organized around three major themes:

- **ML for Networking**  
  Traffic classification, anomaly detection, telemetry analytics, forecasting, routing, congestion control, and resource management

- **AI for Cybersecurity**  
  Intrusion detection, DDoS defense, malware detection, adversarial machine learning, and cyber analytics

- **LLMs for Network Automation and Operations**  
  Log summarization, troubleshooting support, configuration explanation, retrieval-based assistants, and bounded operational copilots

A final cross-cutting theme of the course is:

- **Trustworthy and Secure AI for Networks**  
  Robustness, explainability, calibration, governance, human oversight, and safe deployment
